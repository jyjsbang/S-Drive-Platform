\## 🎯 프로젝트 핵심 목적 (Core Mission)

본 프로젝트는 퍼스널 모빌리티(PM) 시장의 안전 규제 공백을 기술적으로 해결하기 위해 기획되었습니다. 단순히 서비스를 구축하는 것에 그치지 않고, \*\*실시간 주행 데이터를 수집\*\*하고 \*\*Vision AI로 안전 의무(헬멧)를 검증\*\*하여, 추상적인 안전 기준을 \*\*'안전 점수(Safety Score)'\*\*라는 객관적인 수치로 정량화함으로써 \*\*새로운 안전 표준(Standard)\*\*을 정립하는 것이 최종 목적입니다.



\---



\## 🚀 주요 해결 과제 및 기술적 구현 (Key Challenges)



\### 1. \[Data] 주행 데이터 수집 및 위험의 정량화

\* \*\*위험 지표 정립:\*\* '난폭 운전'을 정의하기 위해 NIH(미 국립 보건원) 등 공신력 있는 연구 데이터를 분석하여 급제동(-0.45g), 전복(45도 이상) 등의 실증적 임계값을 설정함.

\* \*\*센서 데이터 활용:\*\* React Native 앱을 통해 가속도계, 자이로스코프, GPS 데이터를 실시간 수집하여 주행 중 발생하는 위험 이벤트를 로그화함.



\### 2. \[AI] Vision AI 기반 안전 의무 검증

\* \*\*헬멧 인증 자동화:\*\* YOLOv8 모델을 활용하여 사용자의 헬멧 착용 여부를 실시간 인증함.

\* \*\*네트워크 최적화:\*\* 고해상도 이미지 전송 시 발생하는 지연을 방지하기 위해, 앱 내부에서 이미지 리사이징 및 압축 후 서버로 전송하는 파이프라인 구축.



\### 3. \[Standard] 데이터 기반 안전 표준(점수) 정립

\* \*\*공정한 점수 산정:\*\* 단순 평균의 한계를 극복하기 위해 거리 가중치와 최신성을 반영한 \*\*거리 가중 이동 평균\*\* 알고리즘을 설계함.

\* \*\*어뷰징 방지:\*\* 짧은 주행으로 점수를 세탁하는 행위를 차단하여, 사용자의 장기적인 운전 습관을 신뢰성 있게 평가함.

\## 🛠 Tech Stack

\* \*\*Backend:\*\* Node.js (Express), MySQL

\* \*\*Frontend:\*\* Vue.js (Admin Dashboard)

\* \*\*Mobile:\*\* React Native (Sensor data collection \& User App)

\* \*\*AI:\*\* Python, YOLOv8



\---



\## 💡 기대 효과

\* \*\*객관적 근거:\*\* 안전 점수를 기반으로 한 보험료 산정 및 이용 제한의 판단 근거 마련.

\* \*\*사회적 비용 감소:\*\* 데이터 기반 정책 수립을 통해 PM 사고 다발 구역 파악 및 인프라 개선 활용.
